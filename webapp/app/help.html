Job notation

Wrapped job group
Asterisk, classified
Light gray, pending
Gray, running
Green, success
Orange, tests failed
Purple, infrastructure exception
Red, build error
Blue, build restarted
Pink, build cancelled
Black, unknown error

Keyboard shortcuts

esc Close all open job or filter panels
j or n Highlight next unclassified failure
k or p Highlight previous unclassified failure
ctrl or cmd Add job to the pinboard during selection
spacebar Add a selected job to the pinboard
r Add a selected job to the pinboard + enter related bug
c Add a selected job to the pinboard + enter classification
ctrl+enter Save pinboard classification and related bugs
ctrl+shift+u Clear the pinboard
i Toggle in-progress (running/pending) jobs
u Show only unclassified failures
(hover) + ctrl/cmd+c Copy logviewer or raw log icon links to clipboard
